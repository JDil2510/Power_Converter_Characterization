# Power Converter Characterization

## Objective

Characterize the electrical performance of commercial DC-DC buck converters through measurements of:

- Output regulation
- Output ripple
- Switching frequency
- Efficiency
- Load behavior

## Equipment

- FNIRSI 1014D Oscilloscope
- FNIRSI DPS-150 Power Supply
